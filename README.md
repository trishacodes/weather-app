🌤️ Weather App

A sleek, modern iPhone-style Weather App built using HTML, CSS, and JavaScript.
Designed with smooth animations, aesthetic UI cards, and dynamic weather visuals like 🌈 rainbows, 🌧️ rain, ⚡ lightning, 🌅 sunrise, and 🌇 sunset.

✨ Features
🌍 Real-time weather data
📍 Location-based weather detection
🌡️ Temperature, humidity, wind speed
🌅 Sunrise & 🌇 Sunset timings
🌧️ Dynamic weather animations:
Rain
Thunderstorms
Clear sky glow
Gradient backgrounds
🎨 iOS-inspired UI:
Glassmorphism cards
Smooth transitions
Premium typography
📱 Fully responsive (mobile + desktop)
🚀 Live Demo


🛠️ Tech Stack
HTML5
CSS3 (Animations + Glassmorphism)
JavaScript (Vanilla JS)
Weather API (e.g. OpenWeatherMap)
